# EXPRESS

Install Node

### Initialize npm

```console
npm init
```

### Initialize Express package

```console
npm i express
```

# NODEMON

```console
npm i nodemon
```

Or global

```console
npm i -g nodemon
```

### Add script to package.json

```json
"scripts": {
    "start": "nodemon app.js"
  },
```

and run server with nodemon

```console
npm start
```
